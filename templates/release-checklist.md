# Publication Release Checklist

- [ ] Canonical website page exists or planned status is explicit.
- [ ] Publication ID and PRRP short route are assigned.
- [ ] Claim boundary is present and not weakened.
- [ ] PDF is final for released artifacts.
- [ ] Manifest, catalog, and checksums are rebuilt.
- [ ] OpenTimestamps receipt is present or clearly pending.
- [ ] DOI/Zenodo URL is present or explicitly unassigned/forthcoming.
- [ ] Verification/status language does not overclaim.
