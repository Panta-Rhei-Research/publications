# PRRP Route Namespace

PRRP routes are typed canonical observatory identifiers such as `https://prrp.site/rn001`.

They are not generic short links. A route ID identifies a released or planned publication artifact class and sequence. The website page remains canonical for human-readable context.

The public route manifest lives at `routes/prrp-routes.public.yml`. It contains only public route records relevant to released or planned publication artifacts.
