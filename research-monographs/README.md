# Research Monographs

This folder records the seven active Second Edition research monographs at book level.

The monographs are retail book artifacts distributed through Amazon/KDP, so this repository does not redistribute manuscript PDFs. Each `book-*` folder contains a GitHub-native publication record with website routes, Corpus routes, Amazon format links, and claim-boundary metadata.

Archived First Edition records and full part/chapter route stubs are intentionally deferred.
