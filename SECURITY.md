# Security Policy

This repository is primarily a publication-artifact mirror, but security still matters.

Please report security concerns involving malicious PDFs, compromised links, spoofed artifacts, workflow vulnerabilities, credential leakage, or supply-chain integrity through the Panta Rhei Research Program contact route:

<hello@panta-rhei.site>

For ordinary broken links, metadata errors, citation issues, or artifact corrections, use repository Issues.
