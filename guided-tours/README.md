# Guided Tours

This folder contains PDF-backed book-level Guided Tour records.

Guided Tours are inspection companions: they name load-bearing hinges, attack vectors, and verification routes for each monograph. They do not replace the books, Corpus Registry, TauLib, Results, or external review.

Each folder contains the PDF copied byte-for-byte from the public website, a `README.md`, a `manifest.json`, and, once stamped, an OpenTimestamps receipt.
